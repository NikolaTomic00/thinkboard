📝Thinkboard – Fullstack Note Taking App
Full-stack application for creating, updating, and deleting notes, built with a focus on scalability and basic backend protection using middleware such as rate limiting to prevent excessive requests and ensure API stability.

![Preview](./assets/preview.png)

🛠️ Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express
Database: MongoDB
Deployment: Vercel

🔗 Live Demo: https://mern-thinkboard-yg2p.onrender.com/

Note: Initial load may take up to 30–60 seconds due to free hosting (Render server cold start).
